# Portal-Entretenimento-Digital
Projeto desenvolvido como parte da **Agenda 13 - CSS** para a disciplina de Desenvolvimento de Sistemas (DS) na ETEC, focado em consolidar a prática de layout e estilização profissional.

O desafio central foi criar um portal temático visualmente chamativo, aplicando as três formas de CSS, o conceito de CSS Reset e, principalmente, o uso de **Flexbox** para um layout ajustável.

## Tecnologias e Conceitos Utilizados

* **HTML5** — Estrutura semântica das seções (Filmes/Séries, Música/Podcasts, Games/Tecnologia).
* **CSS3** — Design Dark Mode com cores neon (Ciano/Aqua) para alto contraste.
* **Flexbox** — Organização das 3 seções em colunas ajustáveis.
* **CSS Reset** — Padronização de estilos entre diferentes navegadores.
* **Responsividade** — Ajuste automático do layout para telas mobile (`flex-wrap`).

## Funcionalidades e Design

* **Layout de 3 Colunas:** Uso de Flexbox (`flex: 1;`) para garantir que todas as seções tenham o mesmo tamanho.
* **Design Impecável:** Garantia de uniformidade visual das imagens ilustrativas (tamanho fixo de 350px x 220px) através do uso de `object-fit: cover;`.
* **Estilização Completa:** Aplicação obrigatória das três formas de CSS (Inline, Interno e Externo).
* **Efeito Hover:** Transição suave nas imagens para maior interatividade.
* **Organização:** Uso consistente de Classes e IDs para estilização modular.

## Aprendizados e Aplicação

Durante o desenvolvimento deste projeto, pratiquei a:

1.  Configuração de layout complexo com **Flexbox** (`display: flex`, `flex-wrap` e `flex: 1`).
2.  Solução para problemas de design com imagens de proporções diferentes (usando `object-fit`).
3.  Implementação de um design moderno com cores contrastantes (Dark Mode e Neon).
4.  Gestão de arquivos de estilo separados (Reset e Estilo principal) e organização do código.
